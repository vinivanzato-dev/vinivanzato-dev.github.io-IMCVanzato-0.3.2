### 🩺 Visão Geral
Dashboard antropométrico profissional focado em Análise de Dados. Essa versão não possui a integração com a geração de um script SQL, possui o gráfico comparativo entre médias de peso.

---

### ⚡ Diferenciais Técnicos v0.3.2
- **📊 Comparativo de Peso**: Gráfico de barras dinâmico que permite visualizar e comparar o peso de toda a base de pacientes.
- **📈 Lógica de Média**: Cálculo em tempo real da média aritmética de peso da clínica via **Python (PyScript)**.
- **🎨 Design Clean & Pro**: Sidebar centralizada, busca minimalista e botão com espaçamento corrigido para maior legibilidade.
- **💾 Full Data Storage**: Persistência completa de Nome, Idade, Peso e Altura.
